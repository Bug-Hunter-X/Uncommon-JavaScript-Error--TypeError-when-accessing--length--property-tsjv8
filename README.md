# Uncommon JavaScript Error: TypeError when accessing 'length' property

This repository demonstrates a common yet subtle error in JavaScript related to accessing the `length` property of a variable. The error arises when attempting to access the `length` property of a variable that isn't an array or a string. 

The file `bug.js` contains the buggy code. The file `bugSolution.js` demonstrates a solution that prevents the error using robust type checking.
